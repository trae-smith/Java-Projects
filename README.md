# Java Projects — COMP 1006 (Carleton University)

Object-oriented programming projects from COMP 1006, 
Winter 2023.

## Projects

### Temperature Converter
`Temperature.java`, `Scale.java`, `SimpleTemperatureProgram.java`

A fully implemented Temperature class supporting Celsius, 
Fahrenheit, and Kelvin scales. Features include:
- Multiple constructors with absolute zero validation
- Scale conversion (setScale converts value automatically)
- Edge case handling for sub-absolute-zero inputs
- toString formatting to 3 decimal places

### Geometric Shape Hierarchy
`Shape.java`, `XYCoord.java`, `Circle.java`, `Square.java`, 
`Triangle.java`, `Quadrilateral.java`, `ShapeExampleApp.java`

An object-oriented shape hierarchy using inheritance. 
Implements area and perimeter calculations for Circle, 
Square, Triangle, and Quadrilateral using XY coordinate 
anchoring. Demonstrates polymorphism and class extension 
in Java.

## Tools
Java, OOP, inheritance, polymorphism
